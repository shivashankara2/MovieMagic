# MovieMagic 🎬
Smart Movie Ticket Booking System

## Features
- User Signup & Login
- Browse Movies
- Seat Booking
- Payment Page
- Booking Confirmation
- User Dashboard

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run the Project
1. Install Python
2. Install requirements
pip install -r requirements.txt

3. Run the app
python app.py
